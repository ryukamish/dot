# My Personal Configuration

## Installation

Many of the configs are inspired from [https://github.com/rwxrob][rwxrob] and [https://github.com/Lukesmithxyz][Luke Smith].

- Directory structure

```
`$HOME`
    |
    -- [.bashrc][`$REPOS`]
        |
        -- github.com
            |
            -- [.bashrc][`$GITUSER`]
                |
                -- (repo name)
```
